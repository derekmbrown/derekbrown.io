Useful little tool to disable the keyboard input on macOS to safely clean the keyboard.

[https://folivora.ai/keyboardcleantool/](https://folivora.ai/keyboardcleantool/)

Can also be installed with Homebrew:

```bash
brew install --cask keyboardcleantool
```
